# MESR-estimation-FISP-and-adoption-of-SAPs
This repo contains Statacode that cleans &amp; manages the data to estimate multinomial endogenous switching regression (MESR) models, as well as the synergies and implications of being a beneficiary of the Farm Input Subsidy Program (FISP) and the adoption of Sustainable Agricultural Practices (SAPs) on agricultural productivity and household welfare.
